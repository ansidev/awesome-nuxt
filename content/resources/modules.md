# Modules

- [SVG Sprite](https://github.com/nuxt-community/svg-sprite-module) - Optimized and Easy way to use SVG files in Nuxt.js.

- [Device](https://github.com/nuxt-community/device-module) - Device detection module for Nuxt.js.
